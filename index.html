Sorotan Proyek

EcoTrack: Dashboard Lingkungan Pintar

[ Tampilan Antarmuka Sistem High-Res Mockup ]
Peran Saya: Fullstack Developer & UI/UX Designer
Garis Waktu: Jan - Mar 2026 (3 Bulan)
Teknologi: Laravel, JavaScript, MySQL, Tailwind CSS

Tujuan Proyek

Menyediakan antarmuka pemantauan real-time bagi organisasi untuk mencatat, mengamati, dan mengurangi indikator emisi karbon secara efisien.

Masalah

Sebagian besar sistem pelacakan masih menggunakan input manual berbasis spreadsheet. Hal ini memicu fragmentasi data dan keterlambatan laporan analisis.

Riset & Desain

Melakukan wawancara pengguna untuk memahami titik kendala data, kemudian menyusun prototipe Figma beresolusi tinggi yang berfokus pada keterbacaan data metrik spike.

Proses Pengembangan

Membangun skema database relasional MySQL, menulis endpoint controller inti pada Laravel untuk menangani input data, dan mengimplementasikan vanilla JavaScript pada sisi klien.

Tantangan & Solusi

Menghadapi masalah performa akibat penulisan database yang terlalu padat secara asinkron. Masalah ini diselesaikan dengan mengaktifkan antrean pemrosesan (queue) bawaan Laravel.

Hasil & Pembelajaran

Berhasil mempercepat kalkulasi laporan emisi sebesar 45%. Proyek ini memberikan pemahaman mendalam tentang pentingnya optimasi indeks database saat menangani skala besar.
